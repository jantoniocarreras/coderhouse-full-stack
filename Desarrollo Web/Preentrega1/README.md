## Contenido Entregado

### 1. Wireframes (Prototipos)
**Archivo:** `wireframes/wireframe-completo.html`

- Wireframes de todas las 5 páginas
- Vista mobile (375px)
- Vista desktop (1200px)

### 2. Estructura HTML
Se entregan 5 archivos HTML completos:

1. **index.html** - Página de inicio con introducción a los sellos frontales y video educativo de YouTube
2. **que-son.html** - Información detallada sobre qué son los sellos
3. **tipos-sellos.html** - Explicación de los 4 tipos de sellos con imágenes reales
4. **beneficios.html** - Beneficios para la salud y sociedad
5. **contacto.html** - Formulario y datos de contacto

#### Características técnicas implementadas:
- ✅ Etiquetas semánticas HTML5 (header, nav, main, section, article, footer)
- ✅ Etiquetas contenedoras (div con clases semánticas)
- ✅ Etiquetas multimedia (img con atributos alt, iframe para video de YouTube)
- ✅ Etiquetas de texto (h1, h2, h3, p, strong)
- ✅ Listas ordenadas y no ordenadas
- ✅ Enlaces de navegación funcionales
- ✅ Formulario de contacto con inputs y textarea
- ✅ Código prolijo con indentación correcta
- ✅ Video de YouTube embebido

### 3. Estilos CSS
**Archivo:** `css/estilos.css`

#### Incluye:
- ✅ **Reset CSS completo** al inicio del archivo
- ✅ Importación de fuente Geist desde Google Fonts
- ✅ Variables CSS para colores consistentes
- ✅ Modificadores de texto (font-family: Geist, font-size, line-height, font-weight)
- ✅ Colores personalizados según paleta definida
- ✅ Estilos para listas (personalizadas y con viñetas)
- ✅ Estilos para encabezados (h1, h2, h3)
- ✅ Estilos para párrafos y textos
- ✅ Estilos para imágenes
- ✅ Estilos para navegación
- ✅ Estilos para formularios
- ✅ Clases de utilidad
- ✅ Código organizado con comentarios

**Nota:** NO se incluyen Box Model ni Flexbox según las indicaciones de la entrega.
